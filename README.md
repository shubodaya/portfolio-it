# Shubodaya Kumar - IT Portfolio

I’m an IT Support Engineer focused on 24x7 enterprise operations, Active Directory, Microsoft 365, and network troubleshooting. This site is my interactive portfolio to share what I do, what I’ve built, and where to reach me.

[![IT Portfolio](https://raw.githubusercontent.com/shubodaya/portfolio-it/main/it.png)](https://github.com/shubodaya/portfolio-it)


- **Live site:** https://shubodaya.github.io/portfolio-it/

## Highlights
- 3D hero and planet scenes built with Three.js, @react-three/fiber, and @react-three/drei.
- Motion-rich UI using Framer Motion plus parallax/tilt effects for projects and credentials.
- Work history rendered with a vertical timeline and bullet-proof points that mirror my resume.
- Certifications, education, and project labs surfaced as cards with quick visual context.
- Contact section includes a Web3Forms-powered form and direct links to LinkedIn/GitHub.

## Tech Stack
- **Framework:** React + Vite
- **Styling:** Tailwind CSS, custom gradients, responsive layout
- **3D & Effects:** three, @react-three/fiber, @react-three/drei, react-parallax-tilt
- **Animation:** Framer Motion, react-vertical-timeline-component
- **Forms:** Web3Forms submission endpoint

## Page Flow
- **Hero:** 3D laptop scene with my intro and role focus.
- **About:** Snapshot of my IT background, LinkedIn/GitHub, and a direct resume link.
- **Services:** What I handle day-to-day—ITSM, endpoint/M365, identity, and networking.
- **Experience:** Timeline for Itarmi IT Services and SonicWall with key incident stats.
- **Skills:** Floating tech spheres for Azure, SIEM, firewalls, VPN, and automation.
- **Projects:** Hands-on labs and automations (Sentinel, AD lab, Python log analyzer) with GitHub shortcuts.
- **Education & Certifications:** MSc Cybersecurity (NCSC certified) plus CCNA, Network+, Security+, AZ-900, CC.
- **Contact:** 3D Earth canvas with a form for direct messages and social buttons.
